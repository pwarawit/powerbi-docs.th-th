# <a name="facebook-connector-for-power-bi-desktop"></a>ตัวเชื่อมต่อ Facebook สำหรับ Power BI Desktop
ตัวเชื่อมต่อ Facebook ใน**Power BI Desktop**อาศัย Facebook Graph API ด้วยเหตุนี้ คุณลักษณะและความพร้อมใช้งานอาจแตกต่างกันเมื่อเวลาผ่านไป

คุณสามารถดู[บทช่วยสอนเกี่ยวกับตัวเชื่อมต่อ Facebook สำหรับ Power BI Desktop](desktop-tutorial-facebook-analytics.md)ได้

Facebook v1.0 ของ Graph API ที่หมดอายุในเดือน 2015 เมษายน 30 Power BI ใช้ API Graph เบื้องหลังสำหรับตัวเชื่อมต่อ Facebook ช่วยให้คุณสามารถเชื่อมต่อและวิเคราะห์ข้อมูลของคุณได้

แบบสอบถามที่สร้างขึ้นก่อนวันที่ 30 2015 เมษายนอาจไม่ทำงาน หรือส่งกลับข้อมูลน้อยลง Subsequent to 30 2015 เมษายน Power BI ได้ v2.8 ในการเรียกใช้ทั้งหมดไปยัง Facebook API ถ้าการสอบถามของคุณสร้างขึ้นก่อนวันที่ 30 เมษายน 2015 และคุณยังไม่ได้ใช้มาตั้งแต่วันที่ดังกล่าว คุณอาจต้องรับรองความถูกต้องอีกครั้งเพื่ออนุมัติชุดสิทธิ์ที่เราจะขอใหม่

แม้ว่าเราพยายามเผยแพร่อัปเดตที่สอดคล้องกับการเปลี่ยนแปลงใด ๆ API อาจเปลี่ยนแปลงในลักษณะที่ส่งผลกระทบต่อผลลัพธ์ของการสอบถามที่เราสร้างขึ้นได้ ในบางกรณี บางแบบสอบถามอาจไม่สนับสนุนอีกต่อไป เนื่องจากการอ้างอิงนี้ เราไม่สามารถรับประกันผลลัพธ์ของแบบสอบถามของคุณเมื่อใช้ตัวเชื่อมต่อนี้

รายละเอียดเพิ่มเติมเกี่ยวกับการเปลี่ยนแปลงใน Facebook API ที่พร้อมใช้งาน[ที่นี่](https://developers.facebook.com/docs/apps/changelog#v2_0)
