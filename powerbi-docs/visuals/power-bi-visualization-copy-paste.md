# <a name="copy-and-paste-a-report-visualization"></a>คัดลอก และวางการแสดงภาพของรายงาน
การคัดลอก และการวางการแสดงภาพจะต้องมีสิทธิ์ในการแก้ไขสำหรับรายงาน ในบริการของ Power BI ซึ่งหมายถึงการเปิดรายงานใน[มุมมองการแก้ไข](../consumer/end-user-reading-view.md) ไม่สามารถคัดลอกและวางการแสดงภาพบนแดชบอร์ดได้

1. เปิดรายงานที่มีการแสดงภาพอย่างน้อยหนึ่งรายการ  

2. เลือกการแสดงภาพ และใช้**Ctrl + C** เมื่อต้องการคัดลอก และ**Ctrl + V** เมื่อต้องการวาง
