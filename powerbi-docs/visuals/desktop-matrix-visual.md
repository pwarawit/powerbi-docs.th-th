# <a name="use-the-matrix-visual-in-power-bi"></a>ใช้เมทริกซ์วิชวลใน Power BI
การ**เมทริกซ์**วิชวลจะคล้ายกับการ**ตาราง**  ตารางสนับสนุน 2 มิติ และข้อมูลที่อยู่แบบแฟลต แสดง และไม่รวมค่าที่ซ้ำกันความหมาย เมทริกซ์ทำให้ง่ายต่อการแสดงข้อมูลมาในทั้งหลายมิติ--สนับสนุนรูปแบบขั้น เมทริกซ์โดยอัตโนมัติรวมข้อมูล และเจาะลงไป 

คุณสามารถสร้างวิชวลเมทริกซ์ใน**Power BI Desktop**และ**บริการ Power BI**รายงานและไฮไลท์ข้ามองค์ประกอบภายในเมทริกซ์กับวิชวลอื่น ๆ บนหน้ารายงานนั้นได้ ตัวอย่าง คุณสามารถเลือกแถว คอลัมน์ และแม้แต่ละเซลล์ และ เน้นแบบไขว้ ยัง แต่ละเซลล์และเลือกเซลล์หลายรายการสามารถคัดลอก และวางลงในแอปพลิเคชันอื่น ๆ 

![ข้ามเมทริกซ์ที่ถูกเน้นและแผนภูมิโดนัท](media/desktop-matrix-visual/matrix-visual_2a.png)

มีคุณลักษณะมากมายที่เกี่ยวข้องกับเมทริกซ์ และเราจะไปศึกษาในส่วนต่อ ๆ ไปของบทความนี้


## <a name="understanding-how-power-bi-calculates-totals"></a>ทำความเข้าใจวิธีที่ Power BI คำนวณผลรวม

ก่อนที่จะกระโดดลงไปในเรื่อง วิธีใช้วิชวล**เมทริกซ์** จำเป็นต้องทำความเข้าใจวิธีที่ Power BI คำนวณค่าผลรวมและผลรวมย่อย ในตารางและเมทริกซ์ สำหรับแถวผลรวมและผลรวมย่อย หน่วยวัดจะคำนวนจากแถวทั้งหมดในข้อมูลเบื้องต้น - ซึ่ง*ไม่*เพียงแค่การบวกค่าในแถวมองเห็นได้ หรือแถวที่แสดงตรง ๆ นี่หมายความว่า คุณอาจได้ค่าผลรวมที่ต่างจากที่คุณคาดหวัง 

ดูที่วิชวลเมทริกซ์ต่อไปนี้ 

![เปรียบเทียบตารางและเมทริกซ์](media/desktop-matrix-visual/matrix-visual_3.png)

ในตัวอย่างนี้ แสดงแต่ละแถวในเมทริกซ์ภาพทางด้านขวาสุด*ยอด*สำหรับแต่ละชุดพนักงานขาย/วัน แต่เนื่องจากพนักงานขายปรากฏในวันที่หลาย ๆ วัน ตัวเลขอาจปรากฏขึ้นมากกว่าหนึ่งครั้ง ดังนั้น ผลรวมที่ถูกต้องจากข้อมูลต้นแบบ กับการบวกง่าย ๆ ของค่าที่มองเห็น จะไม่เท่ากัน นี่คือรูปแบบที่พบบ่อย เมื่อค่าที่คุณกำลังรวมอยู่บนด้าน 'หนึ่ง' ของความสัมพันธ์แบบหนึ่งต่อกลุ่ม

เมื่อดูผลรวมและผลรวมย่อย จำไว้ว่า ค่าเหล่านั้นจะยึดตามข้อมูลเบื้องต้น และไม่ขึ้นกับค่าที่มองเห็นเท่านั้น

## <a name="using-drill-down-with-the-matrix-visual"></a>ใช้เจาะลงกับวิชวลเมทริกซ์
ด้วยวิชวลเมทริกซ์ คุณสามารถทำกิจกรรมที่ไม่พร้อมใช้งานก่อนลึกน่าสนใจมากมาย ซึ่งรวมถึงความสามารถในการดูรายละเอียดแนวลึกที่ระดับแถว คอลัมน์ และแม้แต่ส่วนและเซลล์ ลองมาดูวิธีการดูรายละเอียดแนวลึก

### <a name="drill-down-on-row-headers"></a>การดูรายละเอียดแนวลึกที่ส่วนหัวของแถว
ในบานหน้าต่าง**การแสดงภาพ** เมื่อคุณเพิ่มหลายเขตข้อมูลให้กับส่วน**แถว**ของ**เขตข้อมูล** คุณจะสามารถดูรายละเอียดแนวลึกที่ระดับแถวของภาพเมทริกซ์ได้ ซึ่งจะคล้ายกับการสร้างลำดับชั้นซึ่งจะช่วยให้คุณสามารถดูรายละเอียดแนวลึก (และย้อนกลับ) ตามลำดับชั้นดังกล่าว และวิเคราะห์ข้อมูลในแต่ละระดับได้

ในรูปต่อไปนี้ การ**แถว**ประกอบด้วยส่วน*ขั้นตอนการขาย*และ*ขนาดโอกาส*สร้างจัดกลุ่ม (หรือลำดับชั้น) ในแถวที่เราสามารถดูรายละเอียด

![บัตรตัวกรองแสดงแถวที่จะถูกเลือก](media/desktop-matrix-visual/power-bi-rows-matrix.png)

เมื่อวิชวลนี้มีการจัดกลุ่มที่สร้างขึ้นในส่วน**แถว** วิชวลจะแสดงไอคอน*ดูรายละเอียด*และ*ขยาย* ที่มุมบนซ้ายของวิชวล

![เมทริกซ์ มีตัวควบคุมดูรายละเอียดที่ระบุไว้](media/desktop-matrix-visual/power-bi-matrix-drilldown.png)

คล้ายกับการดูรายละเอียดและขยายดูข้อมูลในภาพอื่น ๆ การเลือกปุ่มเหล่านี้ช่วยให้เราสามารถดูรายละเอียดแนวลึก (หรือย้อนกลับ) ตามลำดับชั้น ในกรณีนี้ เราสามารถเจาะลึกจาก*ขั้นตอนการขาย*เพื่อ*ขนาดโอกาส*ดังที่แสดงในรูปต่อไปนี้ ที่มีการเลือกดูรายละเอียดหนึ่งระดับไอคอน (รูปแฉก) ได้

![เมทริกซ์ มีอธิบายแฉก](media/desktop-matrix-visual/power-bi-matrix-drill3.png)

นอกจากการใช้ไอคอนเหล่านั้น คุณสามารถเลือกส่วนหัวของแถวเหล่านั้น และการดูรายละเอียดแนวลึก โดยเลือกจากเมนูที่ปรากฏ

![ตัวเลือกเมนูสำหรับแถวในเมทริกซ์](media/desktop-matrix-visual/power-bi-matrix-menu.png)

โปรดสังเกตว่า มีหลายตัวเลือกบนเมนูที่ปรากฏ ซึ่งสร้างผลลัพธ์ที่แตกต่างกัน:

เลือก**ลึก**ขยายเมทริกซ์สำหรับ*ที่*แถวระดับ*ยกเว้น*หัวแถวอื่น ๆ ทั้งหมดยกเว้นส่วนหัวของแถวที่เลือก ในรูปต่อไปนี้**ข้อเสนอ** > **ลึก**ถูกเลือก โปรดสังเกตว่า แถวอื่น ๆ ในระดับบนสุดจะไม่ปรากฏในเมทริกซ์ วิธีการดูรายละเอียดนี้เป็นฟีเจอร์ที่มีประโยชน์ และกลายเป็นสิ่งที่น่าสนใจโดยเฉพาะอย่างยิ่งเมื่อเราไปยังส่วน**การไฮไลต์แบบเชื่อมโยง**

![เมทริกซ์เจาะลึกลงหนึ่งระดับ](media/desktop-matrix-visual/power-bi-drill-down-matrix.png)

เลือกคำ**สรุป**ไอคอนเพื่อกลับไปยังมุมมองระดับบนสุดก่อนหน้า ถ้าคุณเลือก**ข้อเสนอ** > **แสดงระดับถัดไป**คุณได้รายการลำดับของรายการระดับถัดไปทั้งหมด (ในกรณีนี้ การ*ขนาดโอกาส*เขตข้อมูล), โดยไม่จัดประเภทลำดับชั้นระดับที่สูงกว่า

![เมทริกซ์ที่ใช้แสดงระดับถัดไป](media/desktop-matrix-visual/power-bi-next-level-matrix.png)

เลือกคำ**สรุป**ไอคอนที่มุมบนซ้ายเพื่อให้เมทริกซ์แสดงประเภทระดับบนสุดทั้งหมด แล้วเลือก**ข้อเสนอ** > **ขยายไปยังระดับถัดไป**ไป ดูค่าทั้งหมดสำหรับทั้งสองระดับของลำดับชั้น - *ขั้นตอนการขาย*และ*ขนาดโอกาส*

![เมทริกซ์โดยใช้การขยายระดับถัดไป](media/desktop-matrix-visual/power-bi-matrix-expand-next.png)

คุณยังสามารถใช้**ขยาย**รายการเมนูการควบคุมการแสดงเพิ่มเติมได้  ตัวอย่าง เลือก**ข้อเสนอ** > **ขยาย** > **เลือก** Power BI แสดงหนึ่งแถวผลรวมสำหรับแต่ละ*ขั้นตอนการขาย*และทั้งหมด*ขนาดโอกาส*ตัวเลือกสำหรับ*ข้อเสนอ*

![เมทริกซ์หลังจากขยายที่นำไปใช้กับข้อเสนอ](media/desktop-matrix-visual/power-bi-matrix-expand.png)

### <a name="drill-down-on-column-headers"></a>ดูรายละเอียดแนวลึกที่ส่วนหัวของคอลัมน์
คล้ายกับความสามารถในการเจาะลึกบนแถว คุณสามารถยังเจาะลึกบน**คอลัมน์**ได้ ในรูปต่อไปนี้ มีสองเขตข้อมูลในการ**คอลัมน์**ด้วยเขตข้อมูล สร้างลำดับชั้นคล้ายกับสิ่งที่เราใช้กับแถวก่อนหน้าในบทความนี้ ในการ**คอลัมน์**เขตข้อมูล เรามี*ภูมิภาค*และ*เซกเมนต์* ทันทีที่ถูกเพิ่มเขตข้อมูลสอง**คอลัมน์**ดรอปดาวน์ใหม่เมนูแสดงบนภาพ ขณะนี้แสดง**แถว**

![เมทริกซ์หลังจากเพิ่มค่าคอลัมน์ที่สอง](media/desktop-matrix-visual/power-bi-matrix-row.png)

เมื่อต้องเจาะลึกบนคอลัมน์ เลือก**คอลัมน์**จากการ*เจาะบน*เมนูที่อยู่มุมบนซ้ายของเมทริกซ์ เลือกคำ*ตะวันออก*ภูมิภาค และเลือก**ลึก**

![เมนูลึกสำหรับคอลัมน์](media/desktop-matrix-visual/power-bi-matrix-column.png)

เมื่อคุณเลือก**ลึก**ระดับของลำดับชั้นคอลัมน์*ภูมิภาค > ตะวันออก*แสดง ซึ่งในกรณีนี้ คือ*จำนวนโอกาส* ภูมิภาคอื่น ๆ แสดง แต่เป็นสีเทา

![ดูเมทริกซ์กับคอลัมน์รายละเอียดหนึ่งระดับ](media/desktop-matrix-visual/power-bi-matrix-column-drill.png)

ส่วนเหลือของรายการเมนูทำงานบนคอลัมน์ในแบบเดียวกับที่พวกเขาทำให้แถว (ดูส่วนก่อนหน้า**เจาะลึกบนส่วนหัวของแถว**) คุณสามารถ**แสดงระดับถัดไป**และ**ขยายไปยังระดับถัดไป**กับคอลัมน์ที่เหมือนกับที่คุณสามารถ มีแถวได้

> [!NOTE]
> ไอคอนการดูรายละเอียดแนวลึกและแนวขึ้นทางมุมบนซ้ายของภาพเมทริกซ์ใช้ได้กับแถวเท่านั้น เมื่อต้องดูรายละเอียดแนวลึกที่ระดับคอลัมน์ คุณต้องใช้เมนูคลิกขวา

## <a name="stepped-layout-with-matrix-visuals"></a>รูปแบบขั้น กับวิชวลเมทริกซ์
วิชวล**เมทริกซ์** ทำการเยื้องประเภทย่อยในลำดับชั้น ใต้แต่ละประเภทใหญ่ ซึ่งเรียกว่า**รูปแบบขั้น**

ในวิชวลเมทริกซ์ เวอร์ชัน*เดิม* ประเภทย่อยจะถูกแสดงในคอลัมน์ต่างหาก และใช้พื้นที่วิชวลมากขึ้น รูปต่อไปนี้แสดงตารางในภาพ**เมทริกซ์**ต้นฉบับ โปรดสังเกตว่า ประเภทย่อยอยู่ในคอลัมน์แยกต่างหาก

![วิธีแบบเก่าของรูปแบบเริ่มต้นสำหรับเมทริกซ์](media/desktop-matrix-visual/matrix-visual_14.png)

ในรูปต่อไปนี้ คุณเห็นวิชวล**เมทริกซ์** ที่แสดงใน**รูปแบบขั้น** โปรดสังเกตว่า ประเภท*คอมพิวเตอร์* มีประเภทย่อย (อุปกรณ์คอมพิวเตอร์ เดสก์ท็อป แล็ปท็อป จอ ฯลฯ) แสดงเยื้องเล็กน้อย ให้วิชวลดูสะอาดตา และดูแน่นขึ้น

![ลักษณะเมทริกซ์ที่จัดรูปแบบข้อมูลปัจจุบัน](media/desktop-matrix-visual/matrix-visual_13.png)

คุณสามารถปรับเปลี่ยนการตั้งค่ารูปแบบขั้นได้อย่างง่ายดาย เมื่อเลือกวิชวล**เมทริกซ์** ในส่วน**รูปแบบ** (ไอคอนลูกกลิ้งสี) ของบานหน้าต่าง**การจัดรูปแบบการแสดงข้อมูล** ขยายส่วน**ส่วนหัวของแถว** คุณมีสองตัวเลือก: การสลับ**รูปแบบขั้น** (ให้เปิดหรือปิด) และ**การเยื้องเค้าโครงแบบขั้น** (ระบุระยะของการเยื้อง เป็นพิกเซล)

![ตัวควบคุมเค้าโครงรูปแบบขั้นแสดงการ์ดส่วนหัวของแถว](media/desktop-matrix-visual/power-bi-stepped-matrix.png)

ถ้าคุณปิด**รูปแบบขั้น** ประเภทย่อยจะแสดงในอีกคอลัมน์ แทนที่จะเยื้องภายใต้ประเภทหลัก

## <a name="subtotals-with-matrix-visuals"></a>ผลรวมย่อย กับวิชวลเมทริกซ์
คุณสามารถเปิดปิดผลรวมย่อย ในวิชวลเมทริกซ์ สำหรับทั้งแถวและคอลัมน์ ในรูปต่อไปนี้ คุณเห็นการตั้งค่าผลรวมย่อยของแถว เป็น**เปิด**

![ผลรวมย่อยและผลรวมแสดงเมทริกซ์](media/desktop-matrix-visual/matrix-visual_20.png)

ในส่วน**จัดรูปแบบ**ของบานหน้าต่าง**การแสดงภาพ** ขยายการ์ด**ผลรวมย่อย**และเปลี่ยนตัวเลื่อน**แถวของผลรวมย่อย**ให้เป็น **ปิด** เมื่อคุณทำเช่นนั้น จะไม่มีแสดงผลรวมย่อย

![เมทริกซ์กับผลรวมย่อยที่ปิดใช้งาน](media/desktop-matrix-visual/matrix-visual_21.png)

กระบวนการเดียวกัน ใช้ได้กับผลรวมย่อยของคอลัมน์

## <a name="cross-highlighting-with-matrix-visuals"></a>การไฮไลต์เชื่อมโยง ด้วยวิชวลเมทริกซ์
ด้วยวิชวล**เมทริกซ์** คุณสามารถเลือกองค์ประกอบใด ๆ ในเมทริกซ์ให้เป็นพื้นฐานสำหรับการไฮไลต์เชื่อมโยงได้ เลือกคอลัมน์ใน**เมทริกซ์** และคอลัมน์นั้นจะถูกไฮไลต์ เช่นเดียวกับวิชวลอื่น ๆ บนหน้ารายงาน ไฮไลต์เชื่อมโยงชนิดนี้เป็นฟีเจอร์ทั่วไปของภาพและการเลือกจุดข้อมูลอื่น ๆ ดังนั้นภาพ**เมทริกซ์**จะให้ฟังก์ชันเดียวกัน

นอกจากนี้ การใช้ Ctrl+คลิก ยังใช้ได้กับการไฮไลต์เชื่อมโยง ตัวอย่างเช่น ในรูปต่อไปนี้ คอลเลกชันของประเภทย่อย ถูกเลือกในวิชวล**เมทริกซ์** สังเกตว่า รายการที่ไม่ได้เลือกในวิชวลจะเป็นสีเทา และวิชวลอื่น ๆ บนหน้าสะท้อนการเลือกในวิชวล**เมทริกซ์**

![รายงานหน้าข้าม highighted โดยเมทริกซ์](media/desktop-matrix-visual/matrix-visual_16.png)

## <a name="copying-values-from-power-bi-for-use-in-other-applications"></a>คัดลอกค่าจาก Power BI เพื่อนำไปใช้ในแอปพลิเคชันอื่น

เมทริกซ์หรือตารางของคุณอาจมีเนื้อหาที่คุณต้องการใช้ในแอปพลิเคชันอื่น เช่น Dynamics CRM, Excel และแม้แต่รายงาน Power BI อื่นๆ คุณสามารถคัดลอกเซลล์แต่ละเซลล์หรือชุดเซลล์หลายเซลล์ลงบนคลิปบอร์ดได้ด้วยการคลิกขวาที่ Power BI จากนั้นวางลงในแอปพลิเคชันอื่น

![ตัวเลือกการคัดลอก](media/desktop-matrix-visual/power-bi-cell-copy.png)

* เมื่อต้องการคัดลอกเซลล์เดียว ให้เลือกเซลล์นั้น คลิกขวา แล้วเลือก**คัดลอกค่า** ในตอนนี้คุณก็สามารถวางค่าที่คัดลอกลงในแอปพลิเคชันอื่นได้ โดยจะได้ค่าเซลล์ที่ไม่ได้จัดรูปแบบในคลิปบอร์ด

    ![ตัวเลือกการคัดลอก](media/desktop-matrix-visual/power-bi-copy.png)

* เมื่อต้องการคัดลอกหลายเซลล์ ให้เลือกช่วงเซลล์ หรือใช้ปุ่ม CTRL เพื่อเลือกเซลล์อย่างน้อยหนึ่งเซลล์ ส่วนคัดลอกจะมีส่วนหัวของคอลัมน์และแถว

    ![วางลงใน Excel](media/desktop-matrix-visual/power-bi-copy-selection.png)

## <a name="shading-and-font-colors-with-matrix-visuals"></a>การแรเงาและสีแบบอักษร กับวิชวลเมทริกซ์
ด้วยวิชวลเมทริกซ์ คุณสามารถนำไปใช้**จัดรูปแบบตามเงื่อนไข**(สีและแถบข้อมูลและการแรเงา) ในพื้นหลังของเซลล์ภายในเมทริกซ์ และคุณสามารถใช้การจัดรูปแบบตามเงื่อนไขกับข้อความและค่าได้

เมื่อต้องจัดรูปแบบตามเงื่อนไข เลือกเมทริกซ์ภาพ และเปิดใน**รูปแบบ**บานหน้าต่าง ขยายการ**จัดรูปแบบตามเงื่อนไข**บัตร และสำหรับ**สีพื้นหลัง**, **สีแบบอักษร**หรือ**แถบข้อมูล**สลับแถบเลื่อนเป็น**บน** เปิดใช้งานหนึ่งในตัวเลือกเหล่านี้แสดงลิงก์สำหรับ*ควบคุมขั้นสูง*ช่วยให้คุณกำหนดสีและค่าสำหรับการจัดรูปแบบสี
  
  ![รูปแบบบานหน้าต่างแสดงตัวควบคุมแถบข้อมูล](media/desktop-matrix-visual/power-bi-matrix-data-bars.png)

เลือก*ควบคุมขั้นสูง*เพื่อแสดงการโต้ตอบ ซึ่งช่วยให้คุณทำการปรับปรุง ตัวอย่างนี้แสดงกล่องโต้ตอบสำหรับ**แถบข้อมูล**

![บานหน้าต่างแถบข้อมูล](media/desktop-matrix-visual/power-bi-data-bars.png)
