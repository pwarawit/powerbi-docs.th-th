# <a name="organizational-custom-visuals-in-power-bi"></a>จัดการแสดงผลด้วยภาพแบบกำหนดเองใน Power BI

คุณสามารถใช้การจัดการแสดงผลด้วยภาพแบบกำหนดเองใน Power BI เพื่อสร้างการแสดงผลด้วยภาพที่เป็นเอกลักษณ์และเหมาะสมกับคุณ การแสดงผลด้วยภาพแบบกำหนดเองเหล่านี้จะถูกสร้างขึ้น โดยนักพัฒนา และพวกเขามักจะสร้างขึ้นเมื่อภาพมากมายที่มีอยู่ใน Power BI ไม่ตอบสนองของพวกเขา

ในบางองค์กร รูปภาพแบบกำหนดเองมีความสำคัญมากยิ่งขึ้น มันอาจะจำเป็นในการสื่อข้อมูลที่เฉพาะเจาะจงหรือข้อมูลเชิงลึกที่มีเอกลักษณ์องค์กร มันอาจมีความต้องการข้อมูลพิเศษหรืออาจจะเน้นวิธการทำีธุรกิจแบบ่วนตัว องค์กรดังกล่าวต้องพัฒนารูปภาพแบบกำหนดเอง แล้วใช้ร่วมกันทั่วทั้งองค์กรและตรวจสอบให้แน่ใจว่า ได้รับการดูแลอย่างเหมาะสม รูปภาพแบบกำหนดเองของ BI power ช่วยให้องค์กรสามารถทำ

รูปต่อไปนี้แสดงขั้นตอนการไหลของวิชวลแบบกำหนดเองขององค์กรใน Power BI จากขั้นตอนผู้ดูแล ผ่านขั้นตอนการพัฒนาและการบำรุงรักษา และไปยังขั้นตอนวิเคราะห์ข้อมูลเป็นลำดับสุดท้าย

![วิชวลแบบกำหนดเอง](media/power-bi-custom-visuals-organizational/custom-visual-org-01.jpg)

การจัดการแสดงผลด้วยภาพขององค์กรมีการปรับใช้ และจัดการ โดยผู้ดูแลระบบ Power BI จากพอร์ทัลผู้ดูแล เมื่อเปิดใช้ในที่เก็บข้อมูลขององค์กร ผู้ใช้ในองค์กรสามารถรู้จักกับมันได้อย่างง่ายดาย และนำเข้าภาพที่กำหนดเองขององค์กรลงในรายงานของพวกเขาได้โดยตรงจาก Power BI Desktop

เพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีการแสดงผลด้วยรูปภาพแบบกำหนดเองขององค์กรในรายงานที่คุณสร้างขึ้น ให้ดูบทความต่อไปนี้: [เรียนรู้เพิ่มเติมเกี่ยวกับการนำภาพองค์กรลงในรายงานของคุณ](power-bi-custom-visuals.md)

## <a name="administer-organizational-custom-visuals"></a>จัดการวิชวลแบบกำหนดเองขององค์กร

เพื่อเรียนรู้เพิ่มเติมเกี่ยวกับวิธีการจัดการ ปรับใช้ และจัดการแสดงผลด้วยภาพแบบกำหนดเองขององค์กรในองค์กรของคุณ ให้ดูบทความต่อไปนี้: [เรียนรู้เพิ่มเติมเกี่ยวกับการปรับใช้และการจัดการแสดงผลด้วยภาพแบบกำหนดเองขององค์กร](https://go.microsoft.com/fwlink/?linkid=866790)

> [!WARNING]
> วิชวลแบบกำหนดเองสามารถมีรหัสซึ่งมีความปลอดภัยหรือความเสี่ยงด้านความเป็นส่วนตัว ตรวจสอบให้แน่ใจว่า คุณเชื่อถือผู้เขียนและแหล่งมาของภาพที่กำหนดเองใด ๆ ก่อนทจะใช้สถานที่จัดเก็บองค์กร

## <a name="considerations-and-limitations"></a>ข้อควรพิจารณาและข้อจำกัด

มีข้อควรพิจารณาและข้อจำกัดที่คุณจำเป็นต้องระวังมากมาย

ผู้ดูแลระบบ

* ไม่รองรับการแสดงผลด้วยภาพกำหนดเองแบบดั้งเดิม (เช่นการแสดงผลด้วยภาพแบบกำหนดเองที่ไม่ได้สร้างขึ้นจาก API รุ่นใหม่)

* เมื่อแบบกำหนดเองจะถูกลบจากเก็บ รายงานใด ๆ ที่มีอยู่ที่ใช้ภาพที่ถูกลบจะหยุดการแสดงผล ไม่สามารถย้อนกลับการลบจากที่เก็บข้อมูลได้ เพื่อปิดการใช้งานรูปภาพแบบรชั่วคราว ให้ใช้คุณลักษณะ "ปิดใช้งาน"

ผู้ใช้ปลายทาง

* วิชวลแบบกำหนดเองขององค์กรเป็นวิชวลส่วนตัว ที่นำเข้ามาจากที่จัดเก็บขององค์กร ด้วยความที่เป็นวิชวลส่วนตัว จึงไม่อาจ[นำออกไปยัง PowerPoint](https://docs.microsoft.com/power-bi/consumer/end-user-powerpoint) หรือแสดงในอีเมลที่รับเข้าเมื่อผู้ใช้[สมัครใช้งานหน้ารายงาน](https://docs.microsoft.com/power-bi/consumer/end-user-subscribe)ได้ เฉพาะ[วิชวลที่กำหนดเองที่ได้รับการรับรอง](https://docs.microsoft.com/power-bi/power-bi-custom-visuals-certified)ที่นำเข้าโดยตรงจาก Marketplace เท่านั้นจึงจะรองรับฟีเจอร์เหล่านี้

* การแสดงผลด้วยภาพของ Visio, ภาพของ PowerApps และภาพของ GlobeMap จาก AppSource Marketplace จะไม่แสดงผล ถ้ามีการเรียกใช้งานผ่านที่เก็บขององค์กร
