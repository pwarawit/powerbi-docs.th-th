# <a name="drill-mode-in-a-visualization-in-power-bi"></a>โหมดการดูรายละเอียดแนวลึกในการแสดงภาพใน Power BI

เอกสารนี้แสดงวิธีการดูรายละเอียดแนวลึกในการแสดงภาพในบริการของ Microsoft Power BI และ Power BI Desktop รายงาน power BI อนุญาตให้ข้อมูลมีหลายลำดับชั้นเพื่อให้คุณรับข้อมูลเชิงลึกสูงสุดในข้อมูลของคุณ ใช้การดูรายละเอียดแนวลึกดูข้อมูลสรุปบนจุดข้อมูลของคุณ คุณสามารถสำรวจรายละเอียดเชิงลึกเกี่ยวกับข้อมูลของคุณ คุณยังสามารถใช้ประโยชน์ได้แม้แต่บนปัจจัยของฟอร์มขนาดเล็กของอุปกรณ์เคลื่อนที่ของคุณ

## <a name="drill-requires-a-hierarchy"></a>การดูรายละเอียดแนวลึก จำเป็นต้องมีลำดับชั้น

เมื่อการแสดงภาพมีลำดับชั้น คุณสามารถเจาะลงไปเพื่อดูรายละเอียดเพิ่มเติม ตัวอย่างเช่น คุณอาจมีการแสดงภาพที่นับจำนวนเหรียญรางวัลโอลิมปิก โดยมีลำดับชั้นที่สร้างขึ้นจาก ชนิดกีฬา ประเภทกีฬา และรายการแข่งขัน ตามค่าเริ่มต้น การแสดงภาพจะแสดงจำนวนเหรียญรางวัลตามชนิดกีฬา - ยิมนาสติก, สกี, กีฬาทางน้ำ และอื่น ๆ แต่เนื่องจากมีลำดับชั้น เลือกองค์ประกอบหนึ่งของการแสดงภาพ (เช่น แท่ง เส้น หรือฟอง) จะแสดงรูปภาพที่มีรายละเอียดเพิ่มขึ้นเรื่อย ๆ เลือกองค์ประกอบ**กีฬาทางน้ำ** เพื่อดูข้อมูลสำหรับประเภทกีฬา ว่ายน้ำ กระโดดน้ำ และโปโลน้ำ  เลือกองค์ประกอบ**กระโดดน้ำ** เพื่อดูรายละเอียดสำหรับการแข่งขัน กระดานสปริงบอร์ด แพลตฟอร์ม และ กระโดดน้ำซิงโครไนซ์

คุณสามารถเพิ่มลำดับขั้นไปยังการรายงานของคุณเองได้ แต่ต้องไม่ใช่การรายงานอื่นที่แชร์กับคุณ
คุณไม่แน่ใจว่าการแสดงภาพ Power BI ไหนมีลำดับชั้น? โฮเวอร์เหนือการแสดงภาพ และถ้าคุณเห็นตัวควบคุมเพื่อดูรายละเอียดเหล่านี้ การแสดงภาพของคุณมีลำดับชั้น

![สกรีนช็อตของไอคอนการดูรายละเอียดเชิงลึกหนึ่งระดับ](./media/end-user-drill/power-bi-drill-icon4.png)  ![สกรีนช็อตของไอคอนเปิดการใช้งานการดูรายละเอียดเชิงลึกหนึ่งระดับ](./media/end-user-drill/power-bi-drill-icon2.png)  ![สกรีนช็อตของไอคอนการดูรายละเอียดลึกในทุกเขตข้อมูลในครั้งเดียว ](./media/end-user-drill/power-bi-drill-icon3.png)
 ![ไอคอนดูข้อมูลสรุป](./media/end-user-drill/power-bi-drill-icon5.png) ![สกรีนช็อตของไอคอนที่ขยายลง](./media/end-user-drill/power-bi-drill-icon6.png)

## <a name="prerequisites"></a>ข้อกำหนดเบื้องต้น

1. เพื่อการทำตาม ให้เปิด[ตัวอย่างวิเคราะห์ร้านค้าปลีก](../sample-datasets.md) สร้างการแสดงภาพ**แผนภูมิต้นไม้**ที่ดู:

    | ---- | ----- |
    | ค่า |ยอดขาย<br>\|\_ จำนวนหน่วยทั้งหมดในปีนี้ |
    | กลุ่ม | จัดเก็บ<br>\|\_ อาณาเขต<br>\|\_ เมือง<br>\|\_ รหัสไปรษณีย์<br>\|\_ ชื่อ

    แผนที่ต้นไม้มีลำดับชั้นที่สร้างขึ้นจากดินแดน, เมือง, รหัสไปรษณีย์ และชื่อ แต่ละดินแดน มีหนึ่งหรือหลายเมือง แต่ละเมือง มีหนึ่งหรือหลายรหัสไปรษณีย์ และต่อไปเรื่อย ๆ ตามค่าเริ่มต้น การแสดงภาพจะแสดงเฉพาะข้อมูลดินแดน เนื่องจาก*ดินแดน*ปรากฏเป็นรายการแรกในลำดับชั้น

    ![สกรีนช็อตของการเรียกบานหน้าต่างการแสดงภาพที่มีเขตข้อมูลออกมา](media/end-user-drill/power-bi-hierarcy-list.png)

1. เรียนรู้เกี่ยวกับความต่างกันของไอคอนดูรายละเอียดพร้อมกันอาจก่อให้เกิดความสับสน กรองแผนที่ต้นไม้เพื่อแสดงเฉพาะสองอาณาเขตที่มีขนาดเล็ก: **KY** และ **TN** เลือกแผนที่ต้นไม้ และภายใต้**ตัวกรองระดับการแสดงภาพ** ขยาย **Territory** แล้วเลือก **KY** และ **TN**

    ตอนนี้ มีเฉพาะสองดินแดนเท่านั้นที่แสดงในแผนที่ต้นไม้

## <a name="three-ways-to-use-the-drill-features"></a>สามวิธีในการใช้คุณลักษณะการดูรายละเอียด

คุณมีหลายตัวเลือกสำหรับการเข้าถึงคุณลักษณะ การดูรายละเอียด ดูข้อมูลสรุป และขยาย สำหรับการแสดงภาพที่มีลำดับชั้น บทความนี้จะแสดงวิธีการใช้ตัวเลือกแรกทางด้านล่าง เมื่อคุณได้เรียนรู้พื้นฐานของการดูรายละเอียดแนวลึกและการขยาย คุณจะทราบวิธีการใช้ทั้งสาม ฟังก์ชันทั้งหมดมีหน้าที่ในสิ่งเดียวกัน ลองใช้งานฟังก์ชันเหล่านั้นดูและเลือกที่คุณชื่นชอบมากที่สุด

- โฮเวอร์เหนือการแสดงภาพเพื่อดู และใช้ไอคอน  

    ![สกรีนช็อตของตัวอย่างโฮเวอร์](./media/end-user-drill/power-bi-hover.png)

- คลิกขวาที่การแสดงภาพเพื่อเปิดเผย และใช้เมนู

    ![สกรีนช็อตของตัวอย่างการคลิกขวา](./media/end-user-drill/power-bi-drill-menu.png)

- จากแถบเมนู Power BI เลือกปุ่ม**สำรวจ**

   ![สกรีนช็อตของตัวเลือกไอคอนสำรวจและดูรายละเอียด](media/end-user-drill/power-bi-explore.png)

## <a name="drill-pathways"></a>เส้นทางการดูรายละเอียด

### <a name="drill-down"></a>ดูรายละเอียดแนวลึก

คุณมีหลายวิธีที่จะดูรายละเอียดการแสดงภาพของคุณ **รายละเอียดแนวลึก**จะนำคุณไปยังลำดับชั้นระดับถัดไป ถ้าคุณกำลังดูระดับ**ดินแดน** คุณสามารถเข้าถึงรายละเอียดในระดับเมือง จากนั้นระดับรหัสไปรษณีย์ และสุดท้ายคือระดับชื่อได้ แต่ละขั้นตอนในเส้นทางจะแสดงข้อมูลใหม่ให้คุณ

![ไดอะแกรมแสดงรายละเอียดของเส้นทาง](./media/end-user-drill/power-bi-drill-path.png)

### <a name="expand"></a>ขยาย

**ขยาย** เป็นการเพิ่มระดับของลำดับชั้นในมุมมองปัจจุบัน หากคุณต้องการดูระดับ**ดินแดน** คุณสามารถขยายและเพิ่มเมือง รหัสไปรษณีย์ และรายละเอียดชื่อไปยังแผนภูมิต้นไม้ได้ แต่ละขั้นตอนในเส้นทางแสดงข้อมูลเดียวกัน และเพิ่มข้อมูลใหม่อีกหนึ่งระดับ

![ไดอะแกรมแสดงการขยายของเส้นทาง](./media/end-user-drill/power-bi-expand-path.png)

คุณยังสามารถเลือกว่าจะดูรายละเอียดแนวลึก หรือขยายทีละหนึ่งเขตข้อมูล หรือทุกเขตข้อมูลพร้อมกันในทีเดียว

## <a name="drill-down-all-fields-at-once"></a>ดูรายละเอียดแนวลึกทุกเขตข้อมูลในครั้งเดียว

1. เริ่มต้นที่ระดับบนสุดของแผนที่ต้นไม้ที่แสดงข้อมูลสำหรับ KY และ TN ขยายทรีแมปของคุณ โดยการเลือกจุดจับจุดหนึ่ง แล้วลากไปด้านขวา

    ![สกรีนช็อตของวิชวลแผนที่ต้นไม้ภาพแสดง KY และ TN](./media/end-user-drill/power-bi-drill-down.png)

1. วิธีดูรายละเอียดแนวลึก*ทุกเขตข้อมูลในครั้งเดียว* ให้เลือกลูกศรคู่ที่มุมซ้ายบนของการแสดงผลภาพ![ไอคอนดูรายละเอียดคู่](./media/end-user-drill/power-bi-drill-icon3.png) แผนที่ต้นไม้ของคุณตอนนี้ แสดงข้อมูลเมืองสำหรับ Kentucky และ Tennessee

    ![สกรีนช็อตของวิชวลแผนที่ต้นไม้ที่แสดงการเจาะรายละเอียดแนวลึกลงไปเมืองภาพ](./media/end-user-drill/power-bi-drill-down1.png)

1. ดูรายละเอียดแนวลึกอีกครั้งสำหรับลำดับชั้นของข้อมูลรหัสไปรษณีย์

    ![สกรีนช็อตของวิชวลแผนที่ต้นไม้ที่แสดงการเจาะรายละเอียดแนวลึกลงไปในรหัสไปรษณีย์](./media/end-user-drill/power-bi-drill-down2.png)

1. หากต้องการดูรายละเอียดสำรอง ให้เลือกลูกศรชี้ขึ้นที่มุมบนซ้ายของการแสดงภาพ ![สกรีนช็อตของไอคอนการดูรายละเอียดเลื่อนขึ้นหนึ่งระดับ](./media/end-user-drill/power-bi-drill-icon5.png).

## <a name="drill-down-one-field-at-a-time"></a>ดูรายละเอียดแนว﻿ลึกทีละเขตข้อมูล

วิธีนี้จะใช้ไอคอนดูรายละเอียด﻿แนวลึกบริเวณมุมบนขวาของตัวการแสดงภาพเอง

1. เลือกไอคอนดูรายละเอียดเพื่อเปิดใช้งานการดูรายละเอียดแนวลึก ![สกรีนช็อตของไอคอนเปิด/ปิดการดูรายละเอียดแนวลึกที่เปิดใช้งานอยู่](./media/end-user-drill/power-bi-drill-icon2.png).

    ในตอนนี้คุณมีตัวเลือกที่จะดูรายละเอียด**ทีละเขตข้อมูล**

    ![สกรีนช็อตของวิชวลที่มีลูกศรชี้ไปที่ไอคอนเปิด/ปิดการดูรายละเอียดเชิงลึกที่เปิดใช้งานอยู่](media/end-user-drill/power-bi-drill-icon-new.png)

    ถ้าคุณไม่เปิดใช้งานการดูรายละเอียด การเลือกองค์ประกอบแสดงภาพ (เช่นแท่ง แผนภูมิฟอง หรือใบไม้) จะไม่เจาะลึก แต่กลายเป็นว่าระบบจะกรองข้ามแผนภูมิอื่นๆ บนหน้ารายงาน

1. เลือกใบไม้สำหรับ **TN** แผนที่ต้นไม้ของคุณจะแสดงเมืองทั้งหมดในรัฐ Tennessee ที่มีร้านค้า

    ![สกรีนช็อตของแผนที่ต้นไม้ที่แสดงข้อมูลสำหรับ TN เท่านั้น](media/end-user-drill/power-bi-drill-down-one1.png)

1. ถึงจุดนี้ คุณสามารถ:

    1. ดูรายละเอียดแนวลึกต่อไปได้สำหรับ Tennessee

    1. ดูรายละเอียดแนวลึกสำหรับเมืองที่ระบุในรัฐ Tennessee

    1. การขยายแทนที่ (ดู**ขยายเขตข้อมูลทั้งหมดในครั้งเดียว**ด้านล่าง)

    ลองดูรายละเอียดแนวลึกทีละเขตข้อมูล  เลือก **Knoxville, TN** (เมือง Knoxville รัฐ Tennessee) แผนที่ต้นไม้ของคุณขณะนี้ แสดงรหัสไปรษณีย์สำหรับร้านค้าของคุณใน Knoxville

    ![สกรีนช็อตของแผนที่ต้นไม้ที่แสดงข้อมูลรหัสไปรษณีย์ 37919](media/end-user-drill/power-bi-drill-down-one2.png)

## <a name="learn-about-the-hierarchy-axis-and-hierarchy-group"></a>เรียนรู้เกี่ยวกับแกนลำดับชั้นและกลุ่มลำดับชั้น

คุณสามารถนึกถึง แกนลำดับชั้นและกลุ่มลำดับชั้น ว่าเป็นกลไกที่คุณสามารถเพิ่มและลด ความละเอียดของข้อมูลที่คุณต้องการดู ข้อมูลใด ๆ ที่สามารถจัดเป็นประเภทและประเภทย่อย สามารถมีลำดับชั้นได้ รวมถึงวันที่และเวลา

คุณสามารถสร้างการแสดงภาพใน Power BI ที่มีลำดับชั้น โดยการเลือกหนึ่งหรือหลายเขตข้อมูล เพื่อเพิ่มไปยัง**แกน**หรือ**กลุ่ม** แล้ว เพิ่มข้อมูลที่คุณต้องการตรวจสอบเป็นเขตข้อมูลใน**ค่า** คุณจะทราบว่าข้อมูลของคุณมีลำดับชั้น ถ้าไอคอน*โหมดดูรายละเอียด* ปรากฎบนมุมบนซ้ายและขวาของการแสดงภาพของคุณ

โดยหลัก ๆ แล้ว สามารถคิดว่าลำดับชั้นของข้อมูลมีสองชนิด:

- ข้อมูลวันที่และเวลา - ถ้าคุณมีเขตข้อมูลชนิด วันที่เวลา คุณมีข้อมูลที่มีลำดับชั้นอยู่แล้ว Power BI สร้างลำดับชั้นสำหรับเขตข้อมูลใด ๆ โดยอัตโนมัติ คุณสามารถแยกวิเคราะห์ค่าลงในการโครงสร้าง[DateTime](https://msdn.microsoft.com/library/system.datetime.aspx)ได้ คุณเพียงแค่เพิ่มเขตข้อมูล วันที่เวลา ไปยัง**แกน**หรือ**กลุ่ม**

- ข้อมูลที่จำแนกประเภท - หากPower BI นำข้อมูลของคุณมาจากคอลเลกชันที่ประกอบด้วยคอลเลกชันย่อย หรือมีแถวข้อมูลที่แชร์ค่าร่วมกัน ถือว่าคุณมีข้อมูลแบบลำดับชั้น

Power BI ให้คุณขยายทีละหนึ่งชุดย่อย หรือชุดย่อยทั้งหมด คุณสามารถดูรายละเอียดข้อมูลของคุณเพื่อดูชุดย่อยเดียวในแต่ละระดับ หรือคุณสามารถดูข้อมูลแนวลึกเพื่อดูทุกชุดย่อยพร้อมกันในแต่ละระดับ ตัวอย่างเช่น คุณสามารถดูรายละเอียดสำหรับปีที่ระบุ หรือดูผลลัพธ์ทั้งหมดสำหรับแต่ละปีเมื่อคุณลงมาตามลำดับชั้น

คุณยังสามารถดูข้อมูลสรุปในลักษณะเดียวกัน

ส่วนต่อไปนี้อธิบายการดูรายละเอียดข้อมูล จากมุมมองสูงสุด มุมมองตรงกลาง และมุมมองต่ำสุด

 เมื่อคุณเจาะลึกลงในรายละเอียด คุณจะเห็นว่ารายงาน**เดี่ยว**มีความเฉพาะเจาะจงมากขึ้น และรายงาน "ดินแดน**ทั้งหมด** มีข้อมูลมากกว่า

## <a name="considerations-and-limitations"></a>ข้อควรพิจารณาและข้อจำกัด

ถ้าการเพิ่มเขตข้อมูลวันที่ ไม่สร้างลำดับชั้น เป็นไปได้ว่าเขตข้อมูลวันที่ ไม่ได้ถูกบันทึกเป็นชนิดวันที่จริง ๆ ถ้าคุณเป็นเจ้าของชุดข้อมูล:

1. เปิดชุดข้อมูลในมุมมอง*ข้อมูล*ใน Power BI Desktop

1. เลือกคอลัมน์ที่มีวันที่

1. ในแท็บ**โมเดล**เปลี่ยน**ชนิดข้อมูล**เป็น**วัน**หรือ**วันที่/เวลา**

ถ้ารายงานมีแชร์ให้กับคุณ ติดต่อเจ้าของเพื่อขอให้เปลี่ยนแปลง
