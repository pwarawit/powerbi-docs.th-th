# <a name="glossary-for-power-bi-service"></a>อภิธานศัพท์สำหรับบริการ Power BI
บริการ Power BI อาจใช้คำศัพท์ที่ไม่คุ้นเคยหรือสับสน อภิธานศัพท์เป็นที่เหมาะสมในการค้นหาคำศัพท์ คุณอาจต้องการเก็บไว้คั่นหน้า อีกแหล่งข้อมูลหนึ่งที่ยอดเยี่ยมสำหรับการเรียนรู้เกี่ยวกับการสร้างการปิดกั้นใน Power BI คือ [แนวคิดพื้นฐาน](end-user-basic-concepts.md) แนวคิดพื้นฐานเป็นภาพรวมระดับสูงเกี่ยวกับ*ส่วนประกอบ*ของ Power BI และความเกี่ยวข้องกัน

อภิธานศัพท์เป็นความพยายามของชุมชน  มองไม่เห็นคำเหรอ โปรดขอให้เราเพิ่มได้ (คุณสามารถใช้ปุ่มคำติชมที่จัดทำเป็นเอกสารด้านล่างของบทความนี้

## <a name="a"></a>A

**บัญชี**Power BI มีบัญชีหลายประเภท  บัญชี Power BI Pro เป็นแบบชำระเงินรายเดือน  บัญชี Power BI ฟรีทำให้สามารถดูเนื้อหาที่อยู่บนความบนจ Power BI Premium  

**พอร์ทัลผู้ดูแลระบบ**ตำแหน่งที่ตั้งที่ผู้ดูแลระบบ Power BI จัดการกับผู้ใช้ คุณลักษณะ และการตั้งค่าสำหรับ Power BI ในองค์กรของพวกเขา (หมายเหตุ: Office 365, Azure และ PowerApps ใช้ศูนย์การจัดการ)

**ผลรวม**เมื่อรวมค่าของหลายแถวเข้าด้วยกันเป็นข้อมูลป้อนเข้าในเกณฑ์เพื่อเป็นรูปแบบค่าเดียวที่มีความหมายหรือการวัดที่สำคัญ

**ผลรวม**ลดแถวในแหล่งข้อมูลพื้นฐานให้พอดีกับแบบจำลอง ผลคือผลรวม

**แจ้งเตือน การแจ้งเตือน**คุณลักษณะที่จะแจ้งให้ผู้ใช้เปลี่ยนแปลงข้อมูลตามขีดจำกัดที่ตั้งไว้ สามารถตั้งค่าการแจ้งเตือนบนไทล์ที่ปักหมุดจากภาพรายงาน ผู้ใช้ได้รับการแจ้งเตือนบนบริการ และแอปสำหรับอุปกรณ์เคลื่อน

**คำอธิบายประกอบ** เขียนข้อความในบรรทัด ข้อความ หรือสแตมป์บนสำเนาสแนปช็อตของไทล์ รายงาน หรือภาพบนแอป Power BI บนอุปกรณ์เคลื่อนที่สำหรับอุปกรณ์ iOS และ Android

**บัตรคำตอบ หน้าคำตอบ**ดูบัตรคำตอบของ Cortana

**แอป แอป**

 
**พื้นที่ทำงานบนแอป**ตอนนี้พื้นที่ทำงานทั้งหมดใน Power BI ยกเว้นพื้นที่ทำงานของฉันเป็นพื้นที่ทำงานบนแอปในตอนนี้ สามารถรวมเนื้อหาในแอป Power BI ที่จัดเก็บไว้ในที่เก็บข้อมูลของ Premium สามารถใช้ร่วมกับผู้ใช้ฟรีได้ พื้นที่ทำงานส่วนบุคคล (ภายใต้พื้นที่ทำงานของฉัน) สามารถเก็บไว้ในที่เก็บข้อมูลของ Premium ได้  

**AppSource**ศูนย์รวมข้อมูลแบบออนไลน์ที่คุณสามารถเรียกดู และค้นหาแดชบอร์ด รายงาน ชุดข้อมูล และแอปเพื่อดาวน์โหลด

**ArcGIS Maps สำหรับ Power BI** ArcGIS เป็นแพลตฟอร์มการทำแผนที่และการวิเคราะห์ที่สร้างขึ้นโดย ESRI ของบริษัท ชื่อของภาพที่อยู่ในคลังข้อมูลภาพของ Power BI เรียกว่า ArcGIS Maps สำหรับ Power BI

**กำหนด** [Power BI Premium] คำกริยาสำหรับย้ายพื้นที่ทำงานไปยังหรือออกจากพื้นที่ทำงานที่กำหนด อย่าใช้การเลื่อนระดับสำหรับการย้ายพื้นที่ทำงานไปยังที่จัดเก็บข้อมูลที่กำหนดเนื่องจากมีความเฉพาะที่เหนือกว่าที่จัดเก็บข้อมูลที่ใช้ร่วมกัน  

## <a name="b"></a>B

**BI ข้อมูลลับทางธุรกิจ**

**ที่คั่นหน้า**มุมมองของข้อมูลที่รวบรวมไว้ในบานหน้าต่างที่คั่นหน้าของรายงานใน Power BI Desktop หรือบริการ จะบันทึกที่คั่นในไฟล์รายงาน.pbix สำหรับการแชร์บนบริการ Power BI ในเดสก์ท็อป 
  
**การแสดงเส้นทาง**การนำทางที่ด้านบนซ้ายเพื่อนำทางระหว่างพื้นที่ทำงานบนแอปอย่างรวดเร็ว

## <a name="c"></a>C

**การคำนวณ**การกำหนดทางคณิตศาสตร์สำหรับขนาดหรือจำนวนของบางอย่าง

**การแสดงผลด้วยภาพแบบกำหนดเองที่ได้รับการรับรอง**การแสดงผลด้วยภาพแบบกำหนดเองของ Power BI ที่เป็นไปตามข้อกำหนดและผ่านการทดสอบความปลอดภัยที่เข้มงวด

**การจัดประเภท**หรือที่เรียกว่าการจัดประเภทข้อมูล ตัวเลือกนี้สามารถเปิดใช้งานได้โดยผู้ดูแลระบบของ Power BI เพื่อสร้างความตระหนักของระดับการรักษาความปลอดภัยที่จำเป็นสำหรับแดชบอร์ด องค์กรสามารถกำหนดชนิดของการจัดแบ่งประเภทได้

**เชื่อมต่อแบบสด**วิธีการเชื่อมต่อกับแบบจำลองข้อมูลบริการวิเคราะห์เซิฟเวอร์ SQL หรืออาจเรียกว่าการเชื่อมต่อแบบสด

**เนื้อหา**


**รายการเนื้อหา**ดัชนีเนื้อหาสำหรับแอป

**ชุดเนื้อหา**คอลเลกชันของชุดข้อมูลแพคเกจล่วงหน้า รายงาน และแดชบอร์ด **แอป**เรียนรู้[เกี่ยวกับการใช้งานพื้นที่ทำงานใหม่](https://docs.microsoft.com/en-us/power-bi/service-create-the-new-workspaces) เลยเดี๋ยวนี้ 

**มุมมองเนื้อหา**

**บัตรคำตอบของ Cortana**หน้ารายงานเฉพาะที่จะรวมไว้ในรายงาน Power BI เพื่อให้สามารถสร้างคำตอบของ Cortana แบบกำหนดเอง

**หน้าคำตอบของ Cortana**ชื่อสำหรับหน้าของ Cortana ที่แสดงต่อลูกค้า

**ตัวกรองข้าม การเน้นข้าม**

**การแสดงผลด้วยภาพแบบกำหนดเอง**ภาพที่สร้างขึ้นโดยชุมชนและ Microsoft สามารถดาวน์โหลดได้จาก Office store สำหรับการใช้งานในรายงานของ Power BI

## <a name="d"></a>D

**แดชบอร์ด**การเลือกการแสดงผลด้วยภาพที่รับส่งข้อมูลสถานะและเมตริกซ์ แดชบอร์ดจัดระเบียบและแสดงข้อมูลด้วยวิธีที่เข้าใจได้ง่าย

**การจัดประเภทข้อมูล**ดูการจัดประเภท

**ตัวเชื่อมต่อข้อมูล**ดูตัวเชื่อม

**รุ่นข้อมูล รุ่นข้อมูลบน Excel**


**ชุดข้อมูล** vs **แหล่งข้อมูล** **ชุดข้อมูล**คือคอลเลกชันของข้อมูลที่ใช้ในการสร้างการแสดงภาพและรายงาน

**ที่เก็บข้อมูลเฉพาะ**[Power BI Premium] แบบจำลองข้อมูลที่ทำงานบนฮาร์ดแวร์เฉพาะในศูนย์ข้อมูลบนคลาวด์ของ Microsoft เพื่อช่วยให้แน่ใจได้ว่ามีประสิทธิภาพการทำงานในระดับที่สม่ำเสมอกัน จะส่งโซลูชัน BI ไปยังอง์กรทั้งหมดโดยไม่คำนึงถึงสิทธิ์การใช้งานของ Power BI

**Desktop**อินสแตนซ์แรก: กล่าวถึง Power BI Desktop Subsequent: เดสก์ท็อป


**ข้าวหลามตัด** Power BI Premium. รูปร่างของไอคอนที่บ่งบอกว่าพื้นที่ทำงานเป็นพื้นที่ทำงานที่มีการจัดเก็บข้อมูลแบบ Premium

**ดูข้อมูลสรุป เจาะลึก เข้าถึงรายละเอียด**

## <a name="e"></a>E

**มุมมองการแก้ไข**โหมดในรายงาน*นักออกแบบ*ที่สามารถสำรวจ ออกแบบ สร้าง และแชร์รายงาน

**จุดไข่ปลา**...

**โค้ดฝังตัว**มาตรฐานทั่วไปบนอินเทอร์เน็ตทั้งหมด ใน Power BI ลูกค้าสามารถสร้างโค้ดฝังตัว และคัดลอกเพื่อวางเนื้อหา เช่น การแสดงผลด้วยภาพในรายงานบนเว็บไซต์หรือบล็อก

**แบบฝังตัว** ดู Power BI Embedded 

**การฝัง** ข้อเสนอของนักพัฒนา Power BI กระบวนการรวมการวิเคราะห์ในแอปโดยใช้ API แบบ REST บน Power BI และ SDK บน Power BI


**สภาพแวดล้อม** [Power BI บนเดสก์ท็อป, Power BI บนอุปกรณ์เคลื่อนที่ ฯลฯ] ใช้สภาพแวดล้อมของ Power BI สำหรับลูกค้าที่มีผู้เช่าหลายราย ใช้คำนี้เพื่ออ้างถึงการเลือกผู้เช่าที่จะเชื่อมต่อ
สามารถใช้สภาพแวดล้อม Power BI (ผู้เช่า) ในเอกสารที่อาจช่วยนักวิเคราะห์ธุรกิจที่คุ้นเคยกับคำที่ผู้เช่าทราบว่าเป็นสิ่งเดียวกัน

## <a name="f"></a>F

**รายการโปรด ยกเลิกรายการโปรด**คำกริยาซึ่งหมายถึงการเพิ่มรายการโปรดเพื่อให้เข้าถึงแดชบอร์ดและรายงานที่เยี่ยมชมบ่อยใน Power BI ได้อย่างรวดเร็ว เมื่อคุณไม่ต้องการให้เป็นรายการโปรดอีกต่อไป ยกเลิกรายการเหล่านั้น

**ตัวกรอง**

**การจัดรูปแบบ**

**โหมดโฟกัส**[บริการบน Power BI]

**บัญชีฟรี**ดูบัญชีในคอลเลกชันของคำนี้

**เต็มหน้าจอ โหมดเต็มหน้าจอ** [บริการบน Power BI]

## <a name="g"></a>G

**เกตเวย์**หรือ**เกตเวย์ข้อมูลภายในองค์กร**สะพานเชื่อมต่อกับแหล่งข้อมูลพื้นฐาน ทำหน้าที่ถ่ายโอนข้อมูลระหว่างบริการของ Power BI และแหล่งข้อมูลภายในองค์กรที่รองรับการรีเฟรชแบบรวดเร็วและปลอดภัย โดยปกติแล้วจะจัดการโดยฝ่ายไอที 

## <a name="h"></a>H
**การแสดงผลด้วยภาพที่มีความหนาแน่นสูง**การแสดงผลด้วยภาพมีจุดข้อมูลมากกว่าที่ Power BI สามารถแสดงออกมาได้ Power BI สุ่มตัวอย่างข้อมูลที่จะแสดงรูปร่างและข้อมูลที่ผิดปกติ

## <a name="i"></a>I

**ข้อมูลเชิงลึก**การวิเคราะห์ข้อมูลโดยอัตโนมัติดำเนินการ โดยการเรียกใช้ข้อมูลเชิงลึกของเราด้วยอัลกอริทึมการเรียนรู้ด้วยเครื่อง

## <a name="k"></a>K

**KPI** ตัวบ่งชี้ประสิทธิภาพหลัก ชนิดของภาพ


## <a name="l"></a>L

**การนำทางด้านซ้าย (นำทางซ้าย)** การควบคุมตามขอบด้านซ้ายของบริการ Power BI

ตัวอย่างแรก: การกล่าวถึงในภายหลังเกี่ยวกับการนำทางด้านซ้ายหรือพื้นที่ว่างที่หนาแน่น: นำทางซ้าย

**หน้ารายการ**หรือ**รายการเนื้อหา**หนึ่งในหน้าส่วนสำหรับองค์ประกอบในการนำทางด้านซ้าย ตัวอย่างเช่น รายการโปรด รายการล่าสุด พื้นที่ทำงานของฉัน ฯลฯ

## <a name="m"></a>M

**แอปสำหรับอุปกรณ์เคลื่อนที่**แอปที่ช่วยให้คุณสามารถเรียกใช้ Power BI บนอุปกรณ์ iOS, Android และ Windows

**การสร้างแบบจำลอง**[Power BI Desktop] รับข้อมูลที่คุณเชื่อมต่อเพื่อพร้อมใช้งานใน Power BI ซึ่งรวมถึงการสร้างความสัมพันธ์ระหว่างตารางในแหล่งข้อมูลหลายแหล่ง การสร้างหน่วยวัด และการกำหนดเมตริก

**พื้นที่ทำงานของฉัน**พื้นที่ทำงานสำหรับลูกค้าแต่ละรายของ Power BI เพื่อสร้างเนื้อหา หากพวกเขาต้องการรวมสิ่งต่อไปนี้สร้างลงในแอป และพวกเขาได้รับอนุญาตสำหรับ*นักออกแบบ* อัปโหลดไปยังพื้นที่ทำงานแอปที่เหมาะสมหรือสร้างใหม่

## <a name="n"></a>N

**ดั้งเดิม** รวมกับผลิตภัณฑ์ ตัวอย่างเช่น Power BI มาพร้อมกับชุดการแสดงผลด้วยภาพแบบ*ดั้งเดิม*  นอกจากนี้คุณยังสามารถนำเข้าแบบประเภทอื่น ๆ ได้ เช่น การแสดงผลด้วยภาพแบบกำหนดเอง

**แผงนำทาง** ดู **การนำทางด้านซ้าย**

**การแจ้งเตือน**ข้อความที่ส่งโดยและถึงศูนย์การแจ้งเตือน Power BI

**Power BI Desktop**หรืออาจเรียกว่า*Desktop* แอปฟรี Windows ของ Power BI คือแอปพลิเคชันฟรีที่คุณสามารถติดตั้งบนคอมพิวเตอร์ของคุณ เพื่อช่วยให้คุณเชื่อมต่อ แปลง และแสดงข้อมูลของคุณ ถูกใช้โดยผู้ออกแบบรายงานและผู้ดูแลระบบ สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [อะไรคือ Power BI](../power-bi-overview.md)

**ใบอนุญาตการใช้งาน Power BI Pro ลิขสิทธิ์**ใบอนุญาตผู้ใช้รายเดือนที่ให้ความสามารถในการสร้างแดชบอร์ด ทำงานร่วมกันในข้อมูลที่แชร์ ปรับปรุงข้อมูลให้ทันสมัยโดยอัตโนมัติ ตรวจสอบและควบคุมการเข้าถึงและใช้ข้อมูล และความสามารถในการจัดชุดเนื้อหาเพื่อแจกจ่าย (แอปบน Power BI)

**Power BI Report Server**การเปิดตัวผลิตภัณฑ์ SSRS ใหม่ด้วย Power BI Premium เพื่อให้องค์กรต่างๆ สามารถสร้างระบบ BI แบบผสมผสานที่กระจาย (การผสมผสานระหว่างการใช้งานระบบคลาวด์และในองค์กร) การฝังตัวทำให้สามารถจัดการเก็บและจัดรายงาน Power BI ให้เป็นภาษาท้องถิ่นได้

**บริการ power BI**บริกาออนไลน์แบบ SaaS (ซอฟต์แวร์ที่เป็นบริการ) สำหรับข้อมูลเพิ่มเติม ให้ดูที่ [อะไรคือ Power BI](../power-bi-overview.md)

**ตัวเชื่อมต่อบริการ Power BI**ตัวเชื่อมต่อนี้ช่วยให้ผู้ใช้ Power BI Desktop สามารถเชื่อมต่อกับฐานข้อมูลในบริการของ Power BI จากเดสก์ท็อป สามารถสร้างแบบจำลองครั้งเดียว และนำกลับมาใช้ในรายงานและแดชบอร์ดมากมาย

## <a name="r"></a>R
**R, Microsoft R**

**มุมมองการอ่าน**มุมมองแบบอ่านอย่างเดียวสำหรับรายงาน (ตรงกันข้ามกับมุมมองการแก้ไข)

**สตรีมแบบเรียลไทม์**ความสามารถในการสตรีมข้อมูลและปรับปรุงแดชบอร์ดแบบเรียลไทม์จากแหล่งข้อมูล เช่น เซนเซอร์ สื่อทางสังคม เมตริกการใช้งาน และสิ่งอื่นจากข้อมูลตามเวลาซึ่งสามารถรวบรวม หรือส่งได้  

**รายการล่าสุด**ที่เก็บในแถบนำทางด้านซ้ายซึ่งเก็บสิ่งที่สร้างขึ้นแต่ละรายการทั้งหมดที่มีการเข้าถึงล่าสุด

**เนื้อหาที่เกี่ยวข้อง**แสดงแต่ละส่วนของเนื้อหาที่สนับสนุนเนื้อหาปัจจุบัน ตัวอย่างเช่น สำหรับแดชบอร์ด คุณสามารถดูรายงานและชุดข้อมูลที่มีข้อมูลและการแสดงภาพบนแดชบอร์ด   

**ลิงก์ที่เกี่ยวข้อง**ลิงก์จากไทล์ของแดชบอร์ดไปยังแดชบอร์ดอื่นและรายงานที่แชร์โดยตรงหรือแจกจ่ายผ่านแอป Power BI ซึ่งช่วยให้มีแดชบอร์ดมากขึ้นที่สนับสนุนการเข้าถึงรายละเอียด

**รายงาน**  
มุมมองแบบหลากหลายมุมมองในชุดข้อมูลเดียว มีการแสดงภาพที่แสดงถึงการค้นพบและข้อมูลเชิงลึกที่แตกต่างกันจากชุดข้อมูลนั้น สามารถมีการแสดงภาพเดียวหรือหลายภาพ หน้าเดียวหรือหลายหน้า  

**ตัวแก้ไขรายงาน**

**รายงานหน่วยวัด**หรือที่เรียกว่าการคำนวณแบบกำหนดเอง Excel เรียกเขตข้อมูลที่คำนวณเหล่านี้ ดูเพิ่มเติมที่*หน่วยวัด*   
**เซิร์ฟเวอร์รายงาน**ตัวอย่างของเซิร์ฟเวอร์รายงาน

**การแสดงภาพที่มีการตอบสนอง** ภาพที่มีการเปลี่ยนแปลงที่ชัดเจนในการแสดงข้อมูลและข้อมูลเชิงลึกให้ได้มากที่สุด โดยไม่ขึ้นกับขนาดของหน้าจอได้
